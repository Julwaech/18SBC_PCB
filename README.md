# 18SBC PCB — Caribou Battery Connector Board

## Overview

The **18SBC** (18S Battery Connector) is the battery connector board for **Project Caribou**, a heavy-lift drone platform by [Arrow Air](https://github.com/Arrow-air).

This PCB interfaces between the battery pack and the aircraft power distribution system.

## Specifications

| Parameter | Value |
|---|---|
| **Series Config** | 18S Li-Ion |
| **Max Voltage** | 75.6V (4.2V × 18) |
| **Nominal Voltage** | 64.8V (3.6V × 18) |
| **Software** | KiCad 10 |
